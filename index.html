Loading Animation

Using keyframes to make a
cool animation

index.html

<div class="loadingAnim"></div>
<div class="loadingAnim2"></div>

style.css:

@keyframes loadingAnimation {
    0% {
        width: 0px;
        transform: translateX(-60px);
        border-color: blueviolet;
    }
    50% {
        width: 30px;
    }
    100%{
        width: 0px;
        transform: translateX(60px);
        border-color: violet;
    }
}

.loadingAnim,
.loadingAnim2
{
    width: 0px;
    height: 0px;
    border: 16px solid blueviolet;
    border-radius: 25px;

    animation-name: loadingAnimation;
    animation-duration: 1s;
    animation-timing-function: ease-in-out;
    animation-direction: alternate;
    animation-iteration-count: infinite;
}

.loadingAnim2{
    animation-direction: alternate-reverse;
}

Hover Transition

Put the mouse on top of the button

index.html

<button class="btn" >Hover Me</button>

style.css:

.btn{
    color: whitesmoke;
    text-align: center;
    position: relative;
    align-self: center;
    width: 80px;
    height: 48px;
    border-width: 3px;
    font-weight: bolder;
    border-radius: 15px;
    margin-bottom: 10px;

    transform: translateY(0%);
    box-shadow: 5px 5px rgba(0, 0, 0, 0.4);
    background-color: rgb(48, 48, 90);
    border-color: slateblue;
    transition: all 1s;
}

.btn:hover{
    border-width: 3px;
    border-color: rgb(48, 48, 90);
    background-color: slateblue;
    transform: translateY(-5px);
}
